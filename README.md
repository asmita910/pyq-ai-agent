# UPSC PYQ AI Agent

## MVP (Minimum Viable Product)

1. Upload UPSC Prelims PYQ PDFs.
2. Extract pure facts from questions (subject-wise & year-wise).
3. Identify topics the user repeatedly forgets.
4. Maintain a progress tracker for daily updates.
5. Generate a 5-fact daily revision list from PYQs.
6. Create weekly analysis once the user provides MCQ responses.
